When to Search Yourself

Cybersecurity requires a "hacker mindset," which relies heavily on independent research, verification, and OSINT (Open Source Intelligence). [[1](https://www.linkedin.com/pulse/what-prerequisite-learn-cyber-security-crawsec-itoic)]

- **Read Official Documentation**: Always read the official man pages (`man chmod`) and documentation (like the Arch Wiki). AI can summarize, but the source text builds exact technical literacy.
- **Stay Up to Date**: Cyber threats change daily. You must monitor real-time threat intelligence feeds, CVE databases, and security blogs (like BleepingComputer or Krebs on Security).
- **Verify Critical Facts**: AI can occasionally hallucinate commands or flags. Always double-check syntax in official docs before running complex commands on live, critical systems. [[1](https://learnwithpride.co.uk/best-tech-skills-to-learn/)]
